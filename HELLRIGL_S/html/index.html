Einleitung

Es soll abstrakter Datentyp Counter implementiert werden, welcher einen Zählers realisiert.

Beispiel

Die Codesequenz um einen Zähler zu bekommen und diesen dann in Fünferschritten von 255 auf 0 zählen zu lassen wäre folgendermaßen:

Counter c = new_counter(DOWN);
set_value(c, 255);
set_increment_value(5);
while (get_value(c) >= 0) {
printf("%d\n", get_value(c));
increment(c);
}

Aufgabe

  1. Erstellen Sie alle notwendigen Komponenten, damit der abstrakte Datentyp (und damit das Gesamtprojekt) baut. Es sollen bis zu acht verschiedene Counter erzeugt werden können. (POS_PRPR_09, POS_PRPR_10, POS_PRPR_11, POS_PRPR_13).
  2. Implementieren Sie die Funktionen von counter so, dass alle Unit Tests erfolgreich durchlaufen (die Descriptors sind bei den einzelnen Unit Tests angegeben).
  3. Refactoren Sie die Funktion init() so, dass Sie die einzelnen Initialisierungsschritte in der Schleife (Eintrag auf frei setzen, increment_value und value auf 0 setzen etc.) in eine eigene Funktion init_one_counter_pool_entry ausgelagert werden. Beachten Sie, dass die Funktion nur innerhalb des Files counter.cpp bekannt sein darf. (POS_PRPR_12, POS_PRPR_04, POS_PRPR_06).
  4. Erklären Sie in einem Kommentar über dem enum Direction, warum an dieser Stelle die Verwendung eines enum der einer symbolischen Konstante vorzuziehen ist (POS_PRPR10).
